  ** $ is used in different ways:-**
1) $xyz to sub  
    name = sugam 
    me $name // me sugam
2) ${} tells explicitly till where to use the variable:-
        var = script
        echo "this is ${var}ing" // this is scripting, $varing would assume varing is the variable.
3) $() runs commnad inside bracket first and then substitute:-
        dir= $(pwd)
        echo "currently in dir"
4) $(()) runs math inside double brackets and treat everything like numbers.
          apple =5
          total = $((apple+10))
          echo "$total" // 15
