functions starting with mosquitto_ are built directly into the Mosquitto # C library #. You did not write these; you are just calling them.

    mosquitto_connect_callback_set(...): A library function.// passed: mosq, on_connect

    mosquitto_subscribe_callback_set(...): A library function.

    mosquitto_message_callback_set(...): A library function.

    mosquitto_loop_forever(...): A library function. This is the engine that keeps your program running endlessly, waiting for network traffic.

    The functions you pass into the library functions are your own code.

    on_connect: You wrote this. You are telling the library, "When you successfully connect to the broker, run my on_connect function so I can print a success log."

    on_subscribe: You wrote this.

    on_message: You wrote this. You are telling the library, "When a message actually arrives, give the raw bytes to my on_message function so I can parse the JSON."
